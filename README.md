# FEG Army Token ($FAT) Telegram Referral Bot

This bot tracks referrals for $FAT and awards points to users.

## Features:
- Unique referral links
- Users earn points when their referrals join & mention them
- `/mylink` to get a referral link
- `/myreferrals` to check referral stats
- `/leaderboard` to view top referrers
